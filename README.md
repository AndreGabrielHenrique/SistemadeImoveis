## Resumo do Projeto

Este repositório contém uma aplicação front-end em React construída com Vite, organizada em componentes reutilizáveis e estilizada com SASS. O projeto fornece uma interface de exemplo para um sistema de listagem de imóveis, com componentes como `Header`, `Banner`, `Card` e `Footer`.

Scripts principais:

- `npm run dev`: inicia o servidor de desenvolvimento.
- `npm run build`: gera a build de produção.
- `npm run preview`: serve a build gerada para revisão.

Para mais detalhes, veja o arquivo de documentação gerado em `docs/Documentacao_Projeto.pdf`.
